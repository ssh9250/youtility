현재 존재하는 django의 models만 참고, views나 serializers는 수정 예정임

홈 화면은 기능 선택 화면
- 5개의 메뉴 선택 가능하게
- 1번째 메뉴 이름은 댓글 유틸리티
- 나머지 4개는 임시로 빈 메뉴, 각각 라우팅 주소는 할당하지만(임시로 a,b,c,d 따위로 대충) 역시 클릭하면 빈 화면만 뜨게끔

1번 메뉴를 클릭하면 주소를 입력할 수 있는 입력창 하나와 크롤 버튼
- 유튜브 주소를 입력하고 크롤 버튼을 누르면 해당 유튜브 영상에 달린 모든 댓글들을 리스트화 해서 화면에 제공
- 화면에 출력할 데이터는 백엔드의 comments 앱의 models 참고
- 댓글의 좋아요가 높은 순서대로 출력
- 20개 단위로 페이징 기능

홈 화면을 제외한 모든 화면에 페이지 리모컨 기능 제공
- 1번 기능으로 바로가기, 2번 기능 바로가기, 홈화면 등 누르면 바로가기 기능 제공


기술스택 요구사항
- react 사용 (vite, 기타 등등 당신이 필요하다고 판단하면 다른 프레임워크도 마음껏 추가/사용)
- 그 외 과설계가 되지 않는 선에서 자유롭게 프레임워크나 패키지 추가 가능
